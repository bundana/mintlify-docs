---
title: "Get API Key"
api: "GET /api-keys/{uuid}"
description: "Retrieve a specific API key by UUID"
---

<ParamField path="uuid" type="string" required>
  The unique identifier (UUID) of the API key
</ParamField>

<Note>
  For security reasons, the full API key is not returned. Only a masked version is shown.
</Note>

<ResponseExample>
```json 200 - Success
{
  "success": true,
  "data": {
    "id": 1,
    "uuid": "f2ed33fd-e4fe-435e-8536-f5f98a155c37",
    "key": "pk_*********************************",
    "status": "active",
    "last_used_at": "2026-01-15T10:00:00.000000Z",
    "created_at": "2026-01-01T00:00:00.000000Z"
  }
}
```

```json 404 - Not Found
{
  "success": false,
  "message": "API key not found."
}
```
</ResponseExample>
