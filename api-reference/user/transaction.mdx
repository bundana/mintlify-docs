---
title: "Get Transaction"
api: "GET /user/transactions/{uuid}"
description: "Retrieve a specific transaction by UUID"
---

<ParamField path="uuid" type="string" required>
  The unique identifier (UUID) of the transaction
</ParamField>

<ResponseExample>
```json 200 - Success
{
  "success": true,
  "data": {
    "id": 1,
    "uuid": "63a13fba-84a4-4fe1-becc-9a305925cf2b",
    "type": "credit",
    "amount": 500.00,
    "status": "completed",
    "description": "Credit purchase - Premium Bundle",
    "reference": "TXN_1767234120056_zetjslzfe",
    "created_at": "2026-01-05T00:00:00.000000Z"
  }
}
```

```json 404 - Not Found
{
  "success": false,
  "message": "Transaction not found."
}
```
</ResponseExample>
