---
title: "Get Message Template"
api: "GET /message-templates/{uuid}"
description: "Retrieve a specific message template by UUID"
---

<ParamField path="uuid" type="string" required>
  The unique identifier (UUID) of the template
</ParamField>

<ResponseExample>
```json 200 - Success
{
  "success": true,
  "data": {
    "id": 1,
    "uuid": "15c981c3-ae58-45b0-8958-39cfbcbe943e",
    "name": "Welcome Message",
    "content": "Hello [First Name]! Welcome to our service.",
    "type": "sms",
    "created_at": "2026-01-01T00:00:00.000000Z",
    "updated_at": "2026-01-01T00:00:00.000000Z"
  }
}
```

```json 404 - Not Found
{
  "success": false,
  "message": "Template not found."
}
```
</ResponseExample>
